unsafe fn shutdown(&self) -> ShutdownFuture<'_>

An optional shutdown function, useful for disconnecting from databases gracefully.

The default impl does nothing

Safety

This should not fail, as it’s ran upon dropping the Gateway, and panicking during a drop means resources haven’t adequately been cleaned up, which isn’t inherintly UB however it should still be documented.
